# Koin basic example
**Technical**
- DI: Koin, Hilt(branch migrate_koin_to_hilt)
- MVVM, LiveData
- Room
- Retrofit

**Main func:**
- Loading movie from api [The Movie DB](https://developers.themoviedb.org/3/movies/get-now-playing).
- Save movie to local database and load to view if have no network
